# Eventitude
An Eventitude-like system where users are able to register and create events, view and update them, comment, vote on comments other people made, and ask questions in any event.

Features the system offers:

Sign Up, Login, Search By Event's Name, Create, Update, Delete And Attend An Event, Upvote Or Downvote A Question, Post A Question, Update And/Or Delete It. Draft Questions And Events, Also Allowing To Update, Delete And Post The Drafted Question(s) And/Or Event(s).
